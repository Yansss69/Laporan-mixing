# Line Sachet Monitor

Aplikasi monitoring produksi untuk efisiensi "Line Sachet" pada sektor consumer goods. 

## Fitur Utama
*   **Tracking Produksi**: Mencatat waktu proses (3m + 12m) secara real-time.
*   **Export Data**: Mengonversi log harian menjadi file Excel (.xlsx) yang rapi.
*   **Responsive UI**: Antarmuka modern dan responsif untuk penggunaan di perangkat mobile (Android/Pydroid).
*   **Data Integrity**: Proteksi input untuk mencegah kesalahan data ganda.

## Spesifikasi Teknis
*   **Language**: Python 3
*   **GUI Framework**: CustomTkinter
*   **Data Processing**: Pandas, OpenPyXL
*   **Platform**: Mobile (Android via Pydroid 3)

## Instalasi
1. Pastikan Pydroid 3 terinstal di perangkat Anda.
2. Instal library yang dibutuhkan melalui Pip:
   `pip install customtkinter pandas openpyxl pillow`
3. Jalankan file `main.py` menggunakan Python 3.
